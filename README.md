# FYW-Node-API

Backend NodeJS - full API.

## Installation

Cloner le dépot

## Usage

Compléter le *.env* grâce au template *cp.env*

Lancer en dev : **yarn dev** ou **npm run dev**
Lancer en production : **yarn start** ou **npm run start**
